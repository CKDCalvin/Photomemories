# Photomemories — Photography Business Website

## Overview

Photomemories is a multi-page photography business website designed to showcase photography services and provide an intuitive way for clients to learn about the business, explore offerings, and submit inquiries or booking requests.

The website was developed using HTML, CSS, and JavaScript, with a focus on clean design, responsive layout, and user-friendly interaction.

---

## Live Demo

🔗 https://photomemoriesv1.netlify.app/

---

## Features

* Multi-page website (Home, Services, Contact, Booking, Summary)
* Responsive layout for desktop, tablet, and mobile devices
* Interactive forms for inquiries and booking
* Form validation using JavaScript
* Dynamic summary page displaying submitted user data
* Navigation system with active page highlighting
* Visual content showcasing photography work

---

## Technologies Used

* **HTML** — Structure and content
* **CSS** — Styling and responsive design
